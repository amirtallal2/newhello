HalloParty - Profile Settings Figma Update
Date: 2026-05-07

What changed
1. زر "الإعدادات" داخل الملف الشخصي يفتح الآن شاشة تحرير الحساب بتصميم الفيجما.
2. الشاشة تعمل باتجاه عربي RTL وتعرض حقول:
   - الاسم
   - الجنس: غير قابل للتعديل
   - عيد ميلاد
   - الدولة الخاصة بك
   - توقيع شخصي
3. تعديل الاسم، تاريخ الميلاد، الدولة، والتوقيع يحفظ على الباك اند الحقيقي عبر API /profile.
4. صورة الحساب يمكن تغييرها من نفس شبكة الصور في الشاشة وتتحفظ على الباك اند الحالي.

Files to use
1. App APK:
   deploy/halloparty_profile_settings_figma_debug.apk

Server update
1. لا يوجد تحديث باك اند لهذا التعديل.
2. لا يوجد تحديث داتا بيز لهذا التعديل.
3. لا ترفع أي ملفات backend بسبب هذا التحديث فقط.

How to test
1. ثبت ملف APK على الموبايل.
2. افتح التطبيق وسجل دخول.
3. افتح الملف الشخصي.
4. اضغط "الإعدادات".
5. عدل الاسم أو تاريخ الميلاد أو الدولة أو التوقيع.
6. ارجع وافتح الشاشة مرة ثانية للتأكد أن البيانات اتحفظت.
